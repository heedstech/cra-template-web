# Heeds Template for Web Applications

Heeds Template for web applications using React with [Create React App](https://create-react-app.dev/).

# Included Libs

- Typescript
- Redux
- Redux Saga
- React Hook Form
- Yup
- React Router Dom
- Axios
- JWT Decode
- Styled Components
- Husky
- React Icons

# Testings Libs

- Jest
- testing-library/react
